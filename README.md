# FinTrack Pro

FinTrack Pro is a personal finance management web application built using **HTML, CSS, and Vanilla JavaScript**. The application allows users to register, log in, and manage their daily income and expenses. It provides a simple dashboard to track financial activities with charts, transaction history, and account settings.

---

# ✨ Features

### 🔐 Authentication

* User Registration
* User Login
* Logout functionality
* Password visibility toggle
* Form validation
* User session stored using LocalStorage

### 📊 Dashboard

* Current Balance
* Total Income
* Total Expenses
* Total Transactions
* Real-time updates after every transaction

### 💳 Transaction Management

* Add new transaction
* Edit existing transaction
* Delete transaction
* Search transactions
* Filter by Income or Expense
* Category-wise transaction management

### 📈 Cash Flow Analysis

* Interactive bar chart using Chart.js
* Displays Income vs Expense
* Updates automatically whenever data changes

### ⚙️ Settings

* Update profile name
* Change currency (USD, EUR, GBP, INR, JPY)
* Dark Mode
* Reset all transaction data

### 🎨 UI Features

* Modern dashboard layout
* Responsive design
* Toast notifications
* Confirmation dialog before deleting
* Clean and user-friendly interface
